# 安装

`liux CLI`需要 [Node.js](https://nodejs.org/) (推荐 `12`及以上版本)。你可以使用 [nvm](https://github.com/creationix/nvm) 或 [nvm-windows](https://github.com/coreybutler/nvm-windows) 在同一台电脑中管理多个 Node 版本。

可以使用下列方法安装这个新的包：

```bash
// 代码库地址：

git clone --recursive 

cd liux-cli

sudo npm install -g
```

安装之后，你就可以在命令行中访问 `liux` 命令。你可以通过简单运行 `liux` ，看看是否展示出了一份所有可用命令的帮助信息，来验证它是否安装成功。

你还可以用这个命令来检查其版本是否正确：

```bash
liux -v
# or
liux --version
```
